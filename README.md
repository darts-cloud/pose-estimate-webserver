# 動画分析アプリ

このアプリは、ユーザーが動画をアップロードし、ポーズ解析を行うためのツールです。Flaskを使用して構築されており、ユーザーは簡単に動画をアップロードし、分析結果をダウンロードできます。

## 特徴

- **動画アップロード**: ユーザーは動画ファイルを選択してアップロードできます。
- **ポーズ解析**: アップロードされた動画に対してポーズ解析を実行し、結果を生成します。
- **プログレスバー**: 動画のアップロード中に進捗状況を表示します。
- **結果のダウンロード**: 分析が完了した後、ユーザーは結果をダウンロードできます。

## 使用技術

- **Flask**: Webアプリケーションフレームワーク
- **OpenCV**: 画像処理ライブラリ
- **YOLO**: オブジェクト検出モデル
- **MediaPipe**: ポーズ推定ライブラリ
- **TQDM**: プログレスバー表示ライブラリ
